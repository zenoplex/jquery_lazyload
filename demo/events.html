Events

getting started

Handle image load/error event.
Open console to see event triggered on lazyload.

$('img.lazy').lazyload({
  load:  function(imageLeft, settings) {
    console.log(this, settings);
    console.log('image left to load: ' + imageLeft);
  },
  error: function(imageLeft, settings) {
    console.alert(this, settings);
    console.log('image left to load: ' + imageLeft);
  }
});
